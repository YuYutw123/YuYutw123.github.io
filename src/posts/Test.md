# 酷酷ㄉ註解
| 格式           | 用途           | 是否被 JavaDoc 工具解析 | 常見用途          |
| ------------ | ------------ | ---------------- | ------------- |
| `/** ... */` | ✅ JavaDoc 註解 | ✅ 會              | 方法／類別／欄位的文件註解 |
| `/* ... */`  | ❌ 普通多行註解     | ❌ 不會             | 一般說明、註解程式碼片段  |
| `// ...`     | 單行註解         | ❌ 不會             | 單行說明          |
